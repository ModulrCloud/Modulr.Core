Modulr.Core dev playground

Development only. Signs envelopes in the browser (Ed25519) and POSTs to POST /message. Open this page from the same host as modulr-core (e.g. http://127.0.0.1:8000/playground/) so requests stay same-origin. Ed25519 signing uses a bundled library served with this app (no CDN required). Key generation uses only your browser’s RNG. If the server rejects a request, the JSON body usually includes status: "error", a code (e.g. PAYLOAD_INVALID), and a detail explaining missing or invalid fields. Canonical JSON for signing matches Python json.dumps (sorted keys, no NaN, string/number rules); unusual floats may still differ in edge cases—use integers in JSON when you can.
